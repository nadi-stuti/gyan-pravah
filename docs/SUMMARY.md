## üé® Key Concepts

### Design System Rules
- **No gradients** - Only solid colors from approved palette
- **Mobile-first** - Responsive design starting from mobile
- **Cultural colors** - Topic-specific color coding
- **Flat design** - Clean, simple aesthetics
- **Touch-friendly** - 44px minimum touch targets

### Technology Stack
- **Next.js 16** with App Router
- **React 19** with TypeScript
- **Zustand** for state management
- **Tailwind CSS 4** for styling
- **Motion** for animations
- **Strapi CMS** for content
- **PostHog** for analytics

### Architecture Principles
- **Component-based** - Reusable, composable components
- **Type-safe** - Full TypeScript coverage
- **Performance-first** - Optimized for mobile devices
- **Accessible** - WCAG compliant design
- **Analytics-driven** - Comprehensive user tracking
